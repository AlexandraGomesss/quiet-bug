# Quiet Bug

**Debug your nerves before the interview.**

Quiet Bug is a live coding interview trainer for junior developers who want to code, communicate and recover with confidence under interview pressure.

## Problem

Many junior developers prepare for interviews by studying code, but freeze during live coding interviews.

Quiet Bug helps them practise not only coding problems, but also structure, communication, panic recovery and soft skills.

## Target User

Junior developers with 0 to 1 year of experience, especially bootcamp graduates, self-taught developers, career changers and junior backend/full-stack candidates.

## MVP Scope

The first version focuses on:

- Java interview exercises
- Thinking checklists
- Panic recovery hints
- Oral explanation practice
- Soft skills prompts
- Employer mindset guidance

## Current Status

Static frontend foundation started.

The project now includes a dependency-free HTML/CSS/JavaScript prototype using static Java exercise data.
